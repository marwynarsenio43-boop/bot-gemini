Guia de Configuração

Siga estes passos para activar o agente no WhatsApp

🚀 PARTE 1 — Colocar o servidor online (Render.com)
1
Criar conta no GitHub
Aceda a github.com e crie uma conta gratuita se ainda não tiver.
2
Fazer upload do projecto
No GitHub, clique + → New repository → Nome: zapbot-cloud → Create.

Depois clique em uploading an existing file, arraste os ficheiros do zip extraído e clique Commit changes.
3
Criar conta no Render
Aceda a render.com → Get Started → Entrar com GitHub.
4
Criar novo Web Service
No Render: New + → Web Service → Selecione o repositório zapbot-cloud → Continue.

Preencha:
• Name: zapbot-cloud
• Runtime: Node
• Build Command: npm install
• Start Command: npm start
• Plan: Free
5
Adicionar variáveis de ambiente no Render
Em Environment Variables, adicione:

GEMINI_KEY = a sua chave Gemini
WA_TOKEN = o token WhatsApp (passo seguinte)
WA_PHONE_ID = o Phone Number ID
VERIFY_TOKEN = zapbot_verify_2024
6
Deploy e obter URL
Clique Create Web Service. O Render faz o deploy automaticamente. Em 2-3 minutos terá um URL como: https://zapbot-cloud.onrender.com
📱 PARTE 2 — Configurar WhatsApp Business API
1
Criar conta Meta for Developers
Aceda a developers.facebook.com e inicie sessão com o Facebook.
2
Criar uma App
My Apps → Create App → Tipo: Business → Next.
Nome: ZapBot → Create App.
3
Adicionar WhatsApp à App
No painel da App, procure WhatsApp → clique Set up. Siga as instruções para adicionar o seu número de telefone comercial.
4
Obter o Token e Phone Number ID
Em WhatsApp → API Setup encontra:
• Temporary access token (válido 24h para testes)
• Phone Number ID

Para token permanente: vá a System Users no Business Manager.
5
Configurar o Webhook
Em WhatsApp → Configuration → Webhook:

• Callback URL: https://zapbot-cloud.onrender.com/webhook
• Verify Token: zapbot_verify_2024

Clique Verify and Save.
Depois em Webhook fields, subscreva: messages
6
✅ Pronto! Testar
Envie uma mensagem WhatsApp para o seu número de teste. O agente deve responder automaticamente em segundos!

Configure o negócio e a persona nas abas ⚙️ Configurar e 🧠 Persona.
💡 Dicas Importantes
• O plano gratuito do Render adormece após 15min sem pedidos. A 1ª mensagem pode demorar 30-60s a acordar.
• Para evitar isso: configure um serviço de ping gratuito em uptimerobot.com para chamar o URL /health a cada 10 minutos.
• O token temporário do Meta expira em 24h. Para produção, crie um token permanente via System Users no Business Manager.
• A API gratuita do WhatsApp permite 1000 conversas/mês gratuitamente.
